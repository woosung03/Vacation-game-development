# DOZI Game Development

## 프로젝트 소개

Unity 기반의 2D 러닝 액션 게임입니다.

플레이어는 장애물을 피하고 맵을 이동하며,
점수를 획득하고 생존하는 것을 목표로 합니다.

본 MD문서는 소프트웨어공학 팀 프로젝트를 기반으로 제작되었으며,
Agile 개발 프로세스를 적용하였습니다.

---

## 개발 목적

- Unity 게임 개발 경험 습득
- GitHub 기반 협업 및 버전관리 경험
- 소프트웨어공학 프로세스 적용
- 팀 프로젝트 기반 개발 경험
